# 💸 Projeto Caixa Eletrônico

Este projeto de um **Caixa Eletrônico virtual** desenvolvido com **HTML, CSS e JavaScript** para praticar lógica de programação, manipulação do DOM e estruturas condicionais.

O sistema simula operações básicas de um caixa eletrônico, permitindo realizar saques e visualizar a quantidade de notas entregues.

---

## 🚀 Acesse o projeto

🔗 [Clique aqui para acessar o site](https://vickmoraisdev.github.io/projeto-caixaeletronico/)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📚 O que pratiquei nesse projeto

- Manipulação do DOM
- Estruturas condicionais
- Conversão de valores
- Lógica de distribuição de notas
- Eventos com JavaScript
- Estilização com CSS
- Alerta para valores abaixo de R$0

---

## 💰 Funcionalidades

- Inserir valor para saque  
- Validação de valores inválidos  
- Cálculo automático das notas entregues  
- Interface simples e responsiva  

---

## 🎯 Objetivo

Esse projeto foi desenvolvido com foco em praticar JavaScript, DOM e melhorar minhas habilidades em lógica de programação através de desafios práticos.
